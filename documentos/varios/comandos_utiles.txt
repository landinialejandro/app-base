Ejecuta este comando desde la raíz del proyecto:

(
find app/Models -type f -name "*.php";
find app/Http/Controllers -type f -name "*.php";
find app/Http/Middleware -type f -name "*.php";
find database/migrations -type f -name "*.php";
find resources/views/orders -type f -name "*.blade.php";
find resources/views/documents -type f -name "*.blade.php";
) | sort | while read file; do
    echo "===== $file ====="
    cat "$file"
    echo
    echo
done > documentos/auditoria/auditoria-tenant.txt


Para crear los archivos de vista:
touch resources/views/documents/index.blade.php
touch resources/views/documents/show.blade.php
touch resources/views/documents/create.blade.php
touch resources/views/documents/edit.blade.php
touch resources/views/documents/_form.blade.php

php artisan optimize:clear
php artisan route:list | grep metrics

limpienza y resembrado
php artisan migrate:fresh --seed

Si querés forzar también limpieza de caches antes
php artisan optimize:clear && php artisan migrate:fresh --seed
